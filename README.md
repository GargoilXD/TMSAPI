# TMSAPI

**Overview:**

This repository contains the TMSAPI backend a Spring Boot application for the Task Management System (TMS). The project structure, tests, and sprint guidance are available in the repository and the attached sprint PDF.

**Sprint Guidance:**

See the sprint document for requirements and acceptance criteria: [TMSSprint.pdf](TMSSprint.pdf)

**Quick links:**

- **Tests & QA:** [TEST.md](TEST.md)
- **Sprint PDF:** [TMSSprint.pdf](TMSSprint.pdf)

**Getting started:**

1. Ensure Java 21 and Maven are installed.
2. Configure your environment variables
3. Build and run:

```bash
mvn -DskipTests=false clean package
java -jar target/*.jar
```

**Running tests & coverage:**

See [TEST.md](TEST.md) for commands, coverage report info and the coverage video link.
